# 🏏 IPL Winning Team Prediction Using Machine Learning

## 📌 Project Overview

This project predicts the winning team of an IPL match using Machine Learning. The model is trained on historical IPL match data and uses match-related features to predict the most likely winner.

## 🎯 Objective

To build a machine learning model that predicts the winning team based on:
- Team 1
- Team 2
- Toss Winner
- Toss Decision
- Match City

## 📂 Dataset

The dataset contains historical IPL match information, including:
- Team 1
- Team 2
- Toss Winner
- Toss Decision
- Match City
- Match Winner

## 🛠️ Technologies Used

- Python
- Pandas
- Scikit-learn
- Random Forest Classifier

## ⚙️ Project Workflow

1. Load the IPL dataset.
2. Clean and preprocess the data.
3. Encode categorical values using Label Encoding.
4. Split the dataset into training and testing sets.
5. Train the Random Forest Classifier.
6. Evaluate the model's performance.
7. Predict the winning team based on user inputs.

## 🚀 How to Run

1. Clone the repository.
2. Install the required libraries:

```bash
pip install pandas scikit-learn
```

3. Place the dataset in the project folder.
